1. Select your zodiac sign from the dropdown
2. Choose a topic you want advice on
3. Type your question

- **Love**: Romantic relationships and love matters

- **Friendship**: Social connections and friendships

## Model Details

- **Base Model**: Microsoft Phi-2 (2.7B parameters)
- **Fine-tuning Method**: LoRA (Low-Rank Adaptation)
- **Quantization**: 4-bit quantization for efficiency
- **Training Data**: Specialized astrology dialogue dataset
- **Training Epochs**: 5
